閩南語常會受國語影響而發錯音，本頁蒐集了一些例子可供參考。

注意：每一組詞語中，顏色相同則發音相同，顏色不同則發音不同。

有底線的字詞，可以點它聽發音。

留意閩南語的變調規則：詞組中非最後一個字需變調。

研gián究    研gián讀≠言giân語≠鉛iân筆    緣iân故

朗讀tho̍k    研讀    閱讀≠中毒to̍k    狼毒    褻瀆

「研讀」的「研」，發音應該跟「研究」的「研」一樣。
而「讀」的聲母則是 ㄊ [th]，不是 ㄉ [t]，讀錯就會變成「毒」或「瀆」。例如「朗讀」常誤讀成「攏毒」。又例如「細讀」這個國語詞組，如果照字讀又讀錯音，就會很像「褻瀆」。

崇拜    崇高≠從來
尊崇≠遵從

近來常聽到「崇」被誤讀成「從」，可能是因為國語的發音相同導致。但兩者音義皆不同，若讀錯則意思改變，例如「尊崇」讀錯就變成「遵從」了。

權力    農曆    經歷≠權利≠建立≠奴隸    美麗

要小心不要把「權力」和「權利」搞混了，兩者的閩南語發音是不同的，意思也大不相同。例如：「你無權力審判別人。」「伊有權利受著公正的審判。」

呼召    徵召    拖吊≠胡椒≠刁≠絕招
召集    呼召罪人    吊車≠招囝婿≠刁持≠招待

「召」這個字常被誤讀成「招」、「刁」或其他的發音，它正確的發音跟「吊」一樣。
其實這個字並不少見，例如兵單或召集令，台語就是「召單」，望遠鏡就是「召鏡」，把畫面拉近放大會說「畫面召近」。還有皇帝「召見」，「召開」記者會等等。
此外，國語的「召ㄓㄠˋ」跟「招ㄓㄠ」發音也是不同的，不少人也會讀錯。

導師    導遊    領導權=道路≠倒店
教導    傳道≠跋倒
報導≠報到

「教導」有時會誤讀為「教倒」，把人「教倒」是不好的。

傳道    傳說    傳染≠攢便便
傳餅及酒≠攢餅及酒(準備)

「傳」的閩南語發音是 ㄊㄨㄢˇ，近年來大眾常受台灣國語的發音影響，而讀成 ㄘㄨㄢˇ 。但是這樣會變成另外一個字：「攢便便」的「攢」。
舉個例：「傳餅及酒」如果讀錯，就變成「攢(準備)餅及酒」，意思跑掉了。

示範    示威≠事務    士兵

「示範」可能受到國語同音字影響，常誤讀為「事範 / 士範」。應該跟「示威抗議」一樣才對。

事物    動物≠事務    任務

「事物」常有人誤讀為「事務」。可能因為國語發音相同不小心口誤了。

敬畏uì    安慰≠礙胃uī
敬畏⁠◻◻    驚畏人    畏寒≠為什麼    胃口

「敬畏⁠◻◻」常有人誤讀為「敬為⁠◻◻」，可想想「畏寒」的發音。

預ī/ū言    預ī/ū告≠寓gū言≠義gī人

「預言」的「預 ī/ū」，並沒有濁音 g，如果讀成 gū，就會變成「寓gū言」，意思完全不同了，真實的變成虛構的。

各kok人    各位≠個koh人    個性
各種各kok樣≠各koh樣

「各人」和「個人」也是常常讀音會互相搞混的詞語，但意思是有很大差別的。例如：「各人造業各人擔」，「這是我個人的見解」。
「各樣」是身體有異樣的意思。

保庇    庇佑    庇護所≠避雨所    避難    避免

「庇」的聲母是 ㄅ，不是 ㄆ。「保庇」沒有人會讀錯，但「庇護所」的「庇」常被誤讀成國語的「屁」，或者是跟「避難所」搞混而亂讀成「避護所」。

強迫    強逼≠氣魄
迫害    迫近    逼近≠魄力

閩南語的「迫」基本上跟「逼」同音（聲母 ㄅ），意義也相近。跟「氣魄」的「魄」（聲母 ㄆ）發音並不同。

考慮lī/lū    憂慮    掛慮≠法律lu̍t=效率lu̍t

不少人會把「憂慮」的「慮」讀成法律的「律」，但奇妙的是「考慮」則沒有人會讀錯。看來是因為台語極少會說「憂慮」這個詞，通常會說「操煩」。
另外，「法律 lu̍t」是短促音第8調，由於北部腔第8調的音高接近第7調，如果「律 lu̍t」沒有發好短促音，就會變成「法慮 lū」。電視上年輕一代演員就常出現此現象。

糾正    糾帶(鬆緊帶)    糾紛=救人≠勼水(縮水)

「糾正」常被誤讀為「勼水」的「勼」，正確發音可以參考「糾帶」、「糾紛」。

敬奉⁠◻◻=奉⁠◻◻⁠的名    奉茶    奉獻

不常用的詞語常會受國語音調影響，例如「敬奉⁠◻◻」、「奉獻」就常受國語影響讀成國語的四聲。可參考其他常用詞調整讀音。

第三章    文章≠緊張    一張

第幾章的「章」，應該跟文章的「章」一樣發音，不少人會誤讀成幾張紙的「張」。

第一節tsat=關節
一節tsat經文    一節tsat甘蔗≠一節tsiat課

「節」這個字用做單位時，有兩種發音：「一節甘蔗」的 tsat 以及「一節課」的 tsiat。差別是：tsat 是用在物品，tsiat 是用在時間。「節」用在經文的單位時，傳統習慣上是讀 tsat。

誘拐=友情
引誘=朋友≠白柚≠手路真幼

「誘」跟「友」同音。從「誘拐」跟「友情」即可知。「引誘」因大眾不熟悉這個詞，常誤讀為「引柚」或「引幼」。

審察    觀察    省察≠審查    調查

順服主人    服從≠祝福新人    福氣    福音

努力≠勞力(謝謝)

「力」這個字有兩種發音，用在不同的詞有各自固定習慣的發音，如果搞混了就可能像這個例子，變成不同的意思。

身上≠神像

「身上」一般習慣的讀法是「身」讀原調，類似「身騎白馬」這樣的讀法。這沒有絕對的規定。但是如果「身」變調的話，就會跟「神像」同音。

身為≠成為

領袖siù    幼秀    野獸siù≠夭壽siū    鳥仔岫(鳥巢)

「野獸」常被誤讀為「野壽」，聽起來像「夭壽」。「領袖」也常被誤讀成「壽」或「岫」。

褻siat瀆    設局≠謝siā禮

「褻瀆」常被誤讀為「謝瀆」。

漏洩≠多謝

宣佈≠通報

任命bīng    誡命≠認命miā

製作tsok    發作    創作≠製造tsō    創造≠種作tsoh    作穡

「製作」跟「製造」音與義皆不同。比如一部電影是用心「製作」的，不是大量「製造」的。因此小心勿把「製作」誤讀成「製造」。

劍≠弓箭    糋雞排的糋

「劍」跟「箭」國語發音相同，閩南語卻不同，小心不要讀錯囉。

評phîng論    評估    批評phîng≠平pîng安

「評」的聲母跟國語一樣是 ㄆ，不是 ㄅ。「評論」常誤讀成「平論」。

改進tsìn≠無盡tsīn≠親近kīn/kūn≠監禁kìm≠受浸tsìm

促進和睦    促進胃腸運動    進步≠盡量

親近⁠◻◻    將近一千萬    近視

「親近⁠◻◻」有時音調會受國語影響，誤讀成國語的四聲。正確發音應該跟「近視」一樣。

實踐    臭賤    手賤≠改建    意見≠推薦    挑戰

「實踐」常誤讀為「實建/見」或「實薦」。因為這幾個字國語同音，不小心會搞混。

溝koo通≠交kau通

以賽亞    馬利亞    何西阿≠文雅    西番雅

「亞」和「雅」也是經常會搞混的兩個字。

所以    所羅門    所多瑪≠鎖門    瑣珥

所羅門    專門≠開門
門徒=文章

小心不要把「所羅門」讀成「鎖⁠◻⁠門」了。「門」有分文讀音及白讀音，人名應該用文讀音，跟「文」一樣發音。

約翰hān≠大漢hàn

路lōo加≠錄lo̍k音

呂底亞    呂布    呂洞賓≠旅行

哈巴谷    山谷    五榖≠講古    七股

經常有人會把「谷」誤讀成「古」，看來是受到國語同音的影響。

約瑟    彩色≠饒赦≠澀

檢索≠線索

拿但業=但是

約旦河=文旦柚

創世記    利未記    筆記≠世紀    年紀

「創世記」的「記」可能是受國語的「記ㄐㄧˋ」或是「世紀」的發音影響，常有人誤讀為國語的 4 聲，正確發音應該跟其他書名一樣。

使徒行傳=不斷

箴tsim言=斟酌

腓立比pí≠筆pit≠威士比
巴比倫    比較≠筆記    必巡(裂縫)

以弗hut所≠佛hu̍t/pu̍t教

歷代志下    志願≠列王記下    記錄

撒督=基督

以掃sò[文音]=糞埽≠摒掃≠嫂    鎖
掃羅=糞埽桶

這裡舉出的是正音讀法，但「以掃」許多人俗唸做「以索」（線索的「索」），「掃羅」則俗唸做「鎖羅」。

摩押=血壓≠鴨

以撒sat=塗虱    狗蝨

敘sū利亞=事先≠旭hiok

拉吉kiat    哈米吉kiat多頓≠攻擊kik≠第一集tsi̍p≠以及ki̍p    普及ki̍p

這幾個字國語都是ㄐㄧˊ，台語發音卻都不同。

攻擊    感激    改革    歌革≠品格

潔kiat淨    結論≠節tsiat目

潔淨tsīng    安靜≠相楗kīng≠保證tsìng

有力≠有利

讀萬卷書    第二卷(文書)≠一卷衛生紙    雞卵卷

內容iông    容允    容易≠龍liông≠榮îng    閒

容易ī    差異ī≠龍椅í≠佮意ì≠回憶ik    一億

「容易」許多人受國語發音影響而讀成「龍椅」，兩個字都讀錯了。

短暫≠短站(短程)

「暫」跟「站」的發音是不同的。如果要照字讀「短暫」，要小心讀成「短站」。但「站」也有時間的意思，如「這站仔」，因此也有人用「短站」來表示短時間的意思。不過，根據教育部辭典，「短站」是短程、短途的意思。

「妝娗」的「娗」因為幾乎沒有同音的字，所以不容易讀，常被誤讀為「妝塔」或「妝疊」。請點一下聽發音。

絆腳石    纏跤絆手=半路    半暝≠拌棉襀被

索引=色彩≠索仔(繩子)

這片phiàn土地≠這片phìnn紙

「片」：大面積或成群的的讀做 phiàn，音同「騙」，例如：「一片厝」(成群的房子)、「歸片攏咧種蕃薯」。小面積的片狀物，才是用 phìnn，如：「一片雪花」、「一片枋仔」。

「裔」這個字也有各種各樣的誤讀，例如「億」。請參考「華裔」的發音，點一下聽聽看。

「訊」這個字跟「信」同音，但因為大家較不熟悉，可能讀成「訓」或「遜」。

鏟除=產業≠攕仔(叉子)

「鏟」這個字跟「產」同音，但因為台語很少說「鏟除」，結果可能受國語發音影響，許多人讀成叉子的「攕」。用叉子是不能把惡勢力鏟除的。

執tsip行    執照    執政≠集tsi̍p合≠職tsit位

「執行」不知為何常誤讀為「集行」或「職行」。但「執照」、「執政」等大家都不會讀錯。

冠軍    冠冕    冠狀病毒=關心≠罐頭
花冠=相關≠一罐

「冠」跟「關」同音，例如「冠軍」跟「關心」。但不熟悉的詞如「冠狀病毒」可能就會讀錯，變成「罐狀病毒」。

恥笑≠齒膏

「恥笑」有時會聽到被誤讀成「齒笑」。

羅網bāng    補破網bāng/羅網bóng[文音]    天羅地網bóng≠拍蠓báng(蚊子)
網路≠蠓蟲

「網」跟蚊子的「蠓 báng」發音是不同的。有首歌叫「補破網 bāng」，熟悉台語的人一定不會讀錯。但不熟悉的詞如「羅網」就可能讀成「羅蠓」。如果是成語「天羅地網」則要文讀 bóng，因此「羅網」也可文讀為 bóng。
另外，「網路」許多人會讀成「蠓路」，但現代 90% 的人都這樣說，已積非成是，約定俗成了。
